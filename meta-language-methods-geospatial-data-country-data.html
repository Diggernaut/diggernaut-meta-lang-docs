Geospatial Data

Country data

You can search for any country by its name and get all the geographical information about it, including the multipolygons of country borders. Also, using data obtained with this command in other commands, you can get the centroid of the country shape. At the moment, you can search only for English name of the country.

This command only works in the geo context and after execution takes you to the page context, where you can go through the XML structure with the data and extract the information you need and save it to the variables or object fields.

    provider: osm
    do:

    - country:
        name: United States of America
        do:
